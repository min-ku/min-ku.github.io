Learning a Vision-Based Footstep Planner for Hierarchical Walking Control on Unstructured Terrain

This project focuses on using vision-based algorithms to generate footstep plans for bipedal robots traversing challenging terrains. Our approach integrates reinforcement learning and hierarchical control to improve adaptability in unstructured environments.

Terrain recognition system to support forestry workers.

A wearable system to support forestry workers that integrated terrain recognition with adaptive control. Featuring a terrain recognition system that classified terrain types and identified safe paths that adjusted control modes in response to different terrains.

AI-based Real-Time Monitoring and Fault Diagnosis for Gear Failure in Electric Vehicle Reducers

This project developed real-time fault diagnosis model with 98% detection, utilizing feature extraction methods such as Wavelet Packet Decomposition, Mel-Frequency Cepstral Coefficients and STFT spectrogram
